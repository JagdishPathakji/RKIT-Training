# 2. HTML Image Tag

## Concepts Covered

- `<img>` Tag
- Image Source (`src`)
- Alternative Text (`alt`)
- Width & Height
- Relative and Absolute Paths
- Accessibility

### Attributes Used

| Attribute | Purpose |
|-----------|----------|
| `src` | Specifies image location |
| `alt` | Alternative text for accessibility |
| `width` | Sets image width |
| `height` | Sets image height |

### Learning Outcome

- Display images on a webpage.
- Use relative and absolute image paths.
- Improve accessibility using the `alt` attribute.
- Resize images using HTML attributes.

# Accessibility Practices Learned

- Use meaningful `alt` text for images.
- Associate every form input with a `<label>`.
- Use semantic HTML elements.
- Use `<th>` instead of `<td>` for table headers.
- Use the `scope` attribute in tables where appropriate.
- Add `<caption>` to describe tables.
